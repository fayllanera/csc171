Steps on how to run the BFS 8-Tile Puzzle

1. Run the following pyhon files on cmd/pycharm:
	
	bfs.py
	state.py
	run.py

2. After running each python files, click the localhost on the run.py file.
	
	localhost:5000

3. You will be then directed to the 8-Tile Puzzle page.

4. Intructions:
   
   By clicking the "Solve" button below the tile, the puzzle will solve       itself.
   On the other hand, the "Refresh" button will refresh the order of the    tiles after it was solve. Then you can click Solve again to solve the    newly ordered tiles.
