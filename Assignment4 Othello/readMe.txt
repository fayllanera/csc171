Steps on how to run the Othello Program

1. Open the file 'requirements.txt' and install the package/s listed.

2. Run the file 'run.py'  on cmd/pycharm:
	
	
2. After running the file, the game window will appear.  

Game Instructions:

	The game is consists of the following players:

	Human vs Computer
	Human vs Human
	Computer vs Computer

	a. You can choose what type of player and/or enemy you want. 
	b. The "Computer Level" function is where the user can 
	   choose the difficulty level of the game.  
	c. After choosing the desired type of players, click "Play".
